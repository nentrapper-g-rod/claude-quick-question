# Quick Question (/q)

A Claude Code plugin that adds a `/q` slash command for asking quick questions without triggering any tools, file edits, or code searches.

## Usage

```
/q what is a Python decorator?
/q how do I reverse a list in JavaScript?
/q what port does PostgreSQL use by default?
```

Claude will answer directly and concisely — no planning, no file edits, no tool use.

## Install

```
/plugin install quick-question
```

Or manually copy `.claude-plugin/` to your project or `~/.claude/`.
